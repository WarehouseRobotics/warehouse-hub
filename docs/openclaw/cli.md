The CLI script `wrobohub-openclaw-cli` (usually linked to /usr/bin) is a wrapper for the original `openclaw` CLI script that allows us to communicate with a running Openclaw gateway via CLI.

If you need to call the same commands from another service or tool without spawning the CLI wrapper directly, see [OpenClaw HTTP Control API](/Users/denis/src/warehouse-hub/docs/openclaw/http-api.md).
